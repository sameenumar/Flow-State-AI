# Mediapipe Models
Place your `.task` model files in this folder. 

### Required Files & Downloads
1. **Face Landmarker**
   - Model: `face_landmarker.task`
   - [Model Documentation & Download](https://ai.google.dev/edge/mediapipe/solutions/vision/face_landmarker/index#models)

2. **Hand Landmarker**
   - Model: `hand_landmarker.task`
   - [Model Documentation & Download](https://ai.google.dev/edge/mediapipe/solutions/vision/hand_landmarker/index#models)

> **TIP**: After downloading, ensure the filenames match the ones listed above exactly so the implementation can find them.
